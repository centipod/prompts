# Centipod Prompts

Free, structured prompts for AI-assisted software engineering — from code review to performance profiling to App Store readiness.

Built and maintained by [Centipod](https://centipod.io).

---

## Prompts

### AI Code Assist Quality Review Prompt
![Version](https://img.shields.io/badge/version-2.1-blue)

A comprehensive prompt that instructs an AI to generate a reusable `quality_review.py` script for any repository. Covers security, privacy, testing, CI/CD, architecture, reliability, and operational readiness — with stack detection, evidence classification, severity-rated findings, baseline/allowlist support, and copy-paste AI fix prompts.

📄 [ai_code_review_prompt.md](ai_code_review_prompt.md)

---

### AI Solution Design Quality Review Prompt
![Version](https://img.shields.io/badge/version-1.0-blue)

A structured prompt for reviewing solution design documents and artefact sets before implementation begins. Covers completeness assessment, architecture quality, requirements coverage, security and privacy design, reliability, scalability, testability, and AI-assist design risks — with evidence-classified findings, severity ratings, and copy-paste improvement prompts.

📄 [ai_design_review_prompt.md](ai_design_review_prompt.md)

---

### Apple Platform App Review Prompt
![Version](https://img.shields.io/badge/version-1.1-blue)

A comprehensive prompt for live review of Apple-platform apps against quality, security, privacy, entitlements, accessibility, App Store compliance, and App Store readiness. Covers iOS, iPadOS, macOS, watchOS, tvOS and visionOS. Produces a structured Markdown report and JSON findings summary with severity-rated findings, executive judgement, and copy-paste AI fix prompts.

📄 [apple_app_review_prompt.md](apple_app_review_prompt.md)

---

### Manual Apple App Store Submission Readiness Review Prompt
![Version](https://img.shields.io/badge/version-1.1-blue)

A prompt for the final human-led review before submitting an Apple-platform app to App Review. Covers App Store metadata, screenshots, privacy disclosures, reviewer notes, business model, account flows, permissions, subscriptions, AI features and policy compliance. Complements automated code review — focuses on what source code alone cannot verify.

📄 [manual-apple-app-store-submission-readiness-review-prompt.md](manual-apple-app-store-submission-readiness-review-prompt.md)

---

### Performance Profiling and Scalability Improvement Review Prompt
![Version](https://img.shields.io/badge/version-1.1-blue)

A structured prompt for periodic performance, scalability and resource-efficiency reviews of software systems. Covers profiling plans, load-test design at normal, peak and 2× peak load, scaling-curve analysis, design-pattern bottlenecks, resource ceilings and regression guardrails — language and platform portable.

📄 [performance-profiling-scalability-improvement-review-prompt.md](performance-profiling-scalability-improvement-review-prompt.md)
